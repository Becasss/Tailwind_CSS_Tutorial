Display - Utilities for controlling the display box type of an element.

When controlling the flow of text, using the CSS property display:inline wil cause the text inside the element to wrap normally.
While using the property display:inline-block will wrap the element to prevent the text from extending beyond its parent.

Lastly, using the property display:block will put the element on its own line and fill its parent.
